Grinnm

Sandbox for a higher-level OCaml-libMagick UI

The goal of this very early sandbox is to provide a higher-level UI for OCaml-libMagick, in order to be the engine of a futur ImageMagick GUI (graphical user interface).

First attempt with assoc-lists

  • grinnm.A.ml
  • grinnm.A.simplified.ml (shorter version, easier to read)
  • result.png (the result of the execution of grinnm.A.ml)
  • different possible file-format syntaxes:
    • syn.xml (close to the MSL)
    • syn.wrl (some peoples find the previous one too verbose)
