Sources for the website Bilingual-eq

OCaml sources for the website bilingual-eq.

Just run make opt to compile the main generator.

This website was built using contents from the Wiktionary and WikiSource.

I'm extracting words equivalences from the wiktionary dump:
- wiktionary-dump/
- wiktionary-dump.xml.bz2

The bilingual books are made from public domain books found on wikiSource.
Making one bilingual book can be made in about one day to several weeks
(or between half an hour and one hour by chapters).
This current version of the web site currently uses 9 bilingual books, included in the archive.

Then the pages content are generated using the 2 small scripts:
- mkpages.ml
- mkindices.ml
