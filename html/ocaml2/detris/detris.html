Detri, a calendar for the console

Detri is a simple program with a command-line user interface written in OCaml that displays a calendar into a console.

There are several versions:

  • detri-mini, a minimal version, easier to extend and/or to read and understand the source code,
  • detri-std, can highlight important dates with colors.
  • detri-range, a variant that can highlight dates and ranges of days with a color.
  • detri-hub, gh's version where people can add issues.
  • detri-m, print w/ a markup language (.htm).
